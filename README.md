# Redux Toolkit Book CRUD App

Welcome to the Redux Toolkit Book CRUD App! This application allows users to perform basic CRUD (Create, Read, Update, Delete) operations on a list of books. It leverages Redux Toolkit for state management, making it easier to manage and update application state.

## Features

- **Add Books:** Add new books to the list.
- **Edit Books:** Modify details of existing books.
- **Delete Books:** Remove books from the list.
- **Update Books:** Save changes to book details.

## Technologies Used

- **React:** JavaScript library for building user interfaces.
- **Redux Toolkit:** A set of tools for efficient Redux state management.
- **React-Redux:** Official React bindings for Redux.
